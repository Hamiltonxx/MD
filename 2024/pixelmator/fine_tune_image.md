# Adjusting colors
Color adjustments are used to change the color, lighting, and contrast in images.  
You can change the overall brightness and contrast of an image, make it look warmer or cooler, or adjust specific colors.  
The color adjustments can be applied to the layers you're editing directly or as separate color adjustments layers.  

# ML Enhance
ML Enhance is an incredibly intelligent and powerful image enhancement feature, like the exposure, white balance, black point, individual color ranges. 
## Automatically improve the look of an image
Format > Color Adjustments > ML Enhance
