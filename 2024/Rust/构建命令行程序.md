Rust的运行速度、安全性、单二进制文件输出和跨平台支持使其成为创建命令行程序的绝佳选择。  

我们来创建minigrep工程，来模拟grep工具。期望运行方式
```shell
$ cargo run searchstring example-filename.txt
```
cargo run 后跟要搜索的字符串以及被搜索的文件路径。

## 读取参数值
```rust
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();
    println!("{:?}", args);

    let query = &args[1];
    let filename = &args[2];
    println!("Searching for {}", query);
    println!("In file {}", filename);
}
```
