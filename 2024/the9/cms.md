# 官网cms
之前是 http://console.cms.the9.com. PHP写的老式cms。
现在是 https://the9admin.bongos.ai/admin  NodeJS写的新式cms。 用的strapi框架。

管理员账号 hamiltonhgz@gmail.com  Aigc2023

已录入目前官网的全部英文和中文的新闻(各20篇左右)

发布新闻用 Content Manager -> Create new entry
编辑好新闻后，可以点击Preview预览。 点击Save保存。
如果即时发布，则点击Publish;如果定时发布，则填写schedule_at字段(不用点击publish),会到点自动发布。
状态是published的新闻，前端可以通过api调用(用户刷新页面就能显示)
