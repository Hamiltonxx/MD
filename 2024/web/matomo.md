# 地址
insight.bongos.ai

# 访问(visit)定义
用户/访客第一次访问网站，或者在网站上30分钟之后的再次操作(期间无操作)，记一次访问(visit)

# 点击(hit)定义
一次点击(hit)可以是 page view, event tracking, download, outlink, onsite search, content tracking request.  
平均一个visit会有3个hits

# 独立访客(unique visitor)定义
独立访客是通过客户端Cookie来定义的，所以不同的终端会统计成不同的独立访客。 而同一访客的多次访问则记为同一个独立访客。  

# page view定义
访问单个页面的事件。在加载或重载页面时，记录页面URL、标题、访问时间等信息。
