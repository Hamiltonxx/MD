近十年来，深度学习改变了很多技术，从图像识别到机器转码。但只有那些获得过专业培训、可以使用专门硬件的人才能解锁深度学习的优势。Fastai框架可以让非专业人士来使用深度学习，从而普及深度学习。它向大众开放深度学习的关键是入门更容易。  
fastai四步:  
1. Create appropriate DataLoaders  
2. Create a Learner  
3. Call a fit method  
4. Make predictions or view results.
