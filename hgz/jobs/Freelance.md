# 背景
在2020年，自由职业(Freelance)占据了美国自由职业的36%，这个数字仍在增长。显然中国也会有越来越多的自由职业者。自由职业的好处很明显，时间自由、项目自由、老板自由、工作量自由。  
随着新冠疫情爆发，越来越多的公司开始招募自由职业者来应对工作需要。供需双方的体量都在不断增长。  
那怎样能开启自由职业呢？通常可以有两张做法。  
1. 创建自己的域名和网站，展示你能提供的服务，然后尽可能的推广。
2. 注册一个或多个freelancer网站。  

这里给出一些最好的freelancer网站。

# Fiverr
类型： 数字营销、web开发、社交媒体  
优点： 种类多、免费注册、在线教程  
缺点： 佣金高、付款流程长  

# Toptal
类型： 软件开发、财务咨询、临时管理  
优点： 顶级公司和人才、免费开票和付款  
缺点： 大规模筛选流程、只有大项目

# Jooble
类型： 写作、平面设计、数据输入  
优点： 工作集中、邮件提醒、无需注册  
缺点： 必须从每个职位空缺来源申请
