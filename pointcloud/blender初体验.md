Blender
# 常用快捷键
删除物体: X 或 fn+delete
# 旋转、缩放、平移
旋转：鼠标中键； 缩放：滚轮； 平移：shift+鼠标中键
# 四视图
Top,Bottom,Left,Right等视图切换: 波浪键
# 移动
点击Move按钮，拖拽x轴或y轴； n键调sidebar,设置位置，一个小格1m； 左下角的微调面板显示刚移动了多少
# 旋转、缩放
旋转按钮，红色、绿色、蓝色圈，最外白色圈，点圈拖； 缩放按钮，拖轴
option+G 坐标归零， option+R 旋转归零， option+S 缩放归零
# 抓取
选中，按g,进入Grab状态，可以任意移动，再按x,y,z轴，沿轴移动
# 复制
cmd+c/cmd+v 原位复制； shift+d 复制物体并进入Grab模式
